Проверка лабораторных

По нажатию кнопки "Проверка лабораторных" Вы попадаете на основную страницу сайта. На этой странице находится таблица с перечнем лабораторных, которые нужно проверить.


Каждая строка содержит поля:

  • Lab - номер лабораторной
  • Student - Имя, фамилия участника
  • Статус - статус лабораторной
  • Отчет - ссылка для перехода на подробный отчет по данной лабораторной

Пример таблицы



Страница отчета

После перехода по ссылке просмотра отчета, открывается страница по конкретной лабе, выполненной участником.

Пример страницы с отчетом



На этой странице, для каждого задания лабораторной, находятся два раздела "Show Diff report for taskX" и "Edit report for taskX".

  • Show Diff report for taskX - отчет по лабораторной с разницей между ответом участника и ответом на лабу.
  • Edit report for taskX - так как в отчет иногда попадают лишние вещи, можно отредактировать его, при желании, и убрать лишнее.


Для удобства, нажав на Show student configs for task1 можно просмотреть конфигурации участника.




Кроме отчетов, на странице находятся:

  • Поле "Add comment" - тут можно написать комментарий к лабе. Особенно это полезно, когда отличий в лабораторной много, но не все из них являются ошибками.
  • Поле "Enter lab mark" - это поле для введения оценки за лабораторную. На странице с отчетом, ниже находятся правила оценки лаб.
  • Кнопка "Done" - После нажания кнопки результаты проверки записываются и ожидают отправки.

При нажатии поля "Show Diff report for taskX", открывается diff отчет:



В отчете

  • команды с минусом - это те команды, которых не было у участника в ответе, но были в правильном ответе лабы (или есть отличия в строке ответа и ответа участника).
  • Команды с плюсом - это то, чего не было в ответе, но что есть в ответе участника.

Для удобства сделана подсветка цветом.



После подсказки о системе оценки, будут находится поля с текстом - это комментарии к уже проверенным лабам (этой же лабе). Часто ошибки одинаковые и так можно скопировать текст комментария и будет быстрее проверять лабы.



Lab info



Страница Lab info позволяет посмотреть initial/answer конфигурации лабораторных, а также diff между initial и answer.



Вы можете посмотреть конфигурации и diff любой лабораторной, для которой уже опубликован ответ и лабораторную, которая назначена для проверки вам (после того, как её проверил Эмиль)


Проверенные лабораторные

Страница Провереные лабораторные позволяет посмотреть какие лабы были проверены. А также, при необходимости, посмотреть еще раз оценку или комментарий и дает возможность исправить их.


Чтобы исправить оценку, достаточно еще раз нажать на ссылку "Перейти к отчету" соответствующей лабы в таблице "Лабораторные проверенные текущим экспертом".

Откроется отчет, с заполнеными полями оценки и комментария (если вы писали комментарий). Тут их можно исправить, при необходимости.


Иногда бывает нужно еще раз сгенерировать отчет по лабе участника. Связанно это может быть с тем, что, например, случайно загружена не та конфигурация

Для перегенерации достаточно выбрать лабу и нажать "Regenerate report"

Обратите внимание, что у вас есть доступ только к тем лабам/отчетам, которые назначены вам
