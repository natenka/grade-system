Help page



По нажатию кнопки "Проверка лабораторных" Вы попадаете на основную страницу сайта. На этой странице находится таблица с перечнем лабораторных, которые нужно просмотреть.


Каждая строка содержит поля:

  • Lab -- номер лабораторной
  • Student -- Имя, фамилия студента
  • Diff -- отличие решения студента от правильного ответа, в процентах
  • Live Test -- результат теста, с выполнением команд show, ping и других
  • Статус -- статус лабораторной
  • Отчет -- ссылка для перехода на подробный отчет по данной лабораторной

Пример таблицы

Lab Student Diff Live Test Статус Отчет
{{ lab }} {{labs[lab]['student']}} {{labs[lab]['diff']}}% {{labs[lab]['live']}} Не проверена Перейти к отчету

После перехода по ссылке просмотра отчета, открывается страница по конкретной лабе, выполненной участником.

На этой странице находятся два раздела "Diff report" и "Live report". Каждый можно развернуть и посмотреть соответствующий отчет.

При необходимости, можно также просмотреть конфигурационные файлы. На странице есть ссылки на конфигурации правильных ответов и конфигурации, которые выложил участник.
